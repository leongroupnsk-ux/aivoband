## 2. Правила перелинковки

- Каждая статья: ≥1 контекстная ссылка на релевантное решение + CTA-блок в конце («Хотите так же → [решение]») + 3 «похожие статьи» из той же рубрики.
- Каждое решение: ссылки на 2–3 поддерживающие статьи («разобрали подробно в блоге») + мини-кейс со ссылкой на /cases.
- Кейс → решение того же типа + CTA на /contacts.
- Хлебные крошки на всех уровнях вложенности (+ BreadcrumbList).

## 3. Технические требования (закладываются на этапе 4)

- **URL:** транслит/латиница, без вложенности глубже 2 уровней: `/blog/chto-takoe-rag`, `/cases/ecommerce-podderzhka`.
- **Микроразметка:** Organization + WebSite (главная) · Service (решения) · FAQPage (все FAQ-блоки) · BlogPosting (статьи) · BreadcrumbList (везде).
- **Мета:** уникальные title ≤ 60 зн. / description ≤ 160 зн. на каждую страницу; OG-шаблон (обложка + заголовок + лого Aivo).
- **Sitemap/robots:** next-sitemap с авто-обновлением при публикации; canonical на всех страницах; RSS для блога.
- **Перформанс как SEO-фактор:** LCP < 2.5с, CLS < 0.1 — бюджет закреплён в критериях приёмки.
